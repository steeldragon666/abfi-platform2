### Core Concepts

1. **Futures Marketplace**: Long-term commitment mechanism for perennial crops (up to 25 years)
2. **Critical Infrastructure**: Platform as essential market coordination layer
3. **Network Effects**: Value increases with participation
4. **Stakeholder Alignment**: Optimising for ecosystem value, not extraction
5. **Continuous Evolution**: Commitment to ongoing development

### Key Differentiators

- **Forward Contracting**: Enabling investment in perennial energy crops
- **Verification Infrastructure**: Trust layer for counterparty assessment
- **Price Transparency**: Market-based price discovery
- **Data Generation**: Industry intelligence for all participants
